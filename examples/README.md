# Hywrapper Examples

This directory contains example projects demonstrating how to use the Hywrapper library in various environments.

## Python
- `pyproject.toml`: Modern Python project configuration.
- `requirements.txt`: Standard pip dependency list.
- `main.py`: Example usage with `asyncio`.

## TypeScript
- `package.json`: NPM package configuration.
- `tsconfig.json`: TypeScript compiler configuration.
- `src/main.ts`: Type-safe example usage.

## JavaScript (ESM)
- `package.json`: NPM package configuration with `"type": "module"`.
- `main.js`: JavaScript example usage.

## Kotlin
- `build.gradle.kts`: Kotlin DSL Gradle build script.
- `src/main/kotlin/com/example/Main.kt`: Idiomatic Kotlin example with coroutines.

## Java
- `build.gradle`: Groovy DSL Gradle build script.
- `src/main/java/com/example/Main.java`: Java example showing interoperability with Kotlin coroutines.
